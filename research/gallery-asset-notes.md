# Набор изображений для фотогалереи услуг

После просмотра отобранных изображений подтверждено, что для галереи подходят как минимум два визуальных направления.

Первое — фото внутренней части колодца с водой и техническими элементами. Оно хорошо работает для блока по чистке колодцев: изображение читается как реальный рабочий процесс, имеет глубину и поддерживает индустриальный характер интерфейса.

Второе — фото узла водоснабжения в помещении с баком и медными трубами. Оно подходит для раздела по подводке воды в дом: сцена инженерная, предметная и визуально близка к общей тёмно-премиальной подаче сайта.

Следующим шагом нужно собрать итоговый комплект по четырём типам услуг: чистка, ремонт, углубление и водоснабжение, затем загрузить выбранные файлы в webdev-CDN и связать их с контентной моделью сайта.

Архитектура галереи:

- Общий блок на страницах услуг с фильтрами: «Все работы», «Чистка», «Ремонт», «Углубление», «Водоснабжение».
- Карточки с изображением, названием кейса, услугой, локацией и коротким результатом.
- На каждой сервисной странице галерея должна автоматически поднимать релевантные элементы текущей услуги в начало выдачи.
- Для сохранения масштаба и ритма интерфейса базовый набор лучше собрать из 8 карточек: по 2 примера на каждую услугу.
